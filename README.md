# Personal-Trainer-Lading-Page
Proyecto de landing page para mi portfolio. Presenta un servicio de entrenador personal con enfoque en rendimiento, método, servicios, programas y contacto. El objetivo es mostrar diseño visual, jerarquía de contenido y una estructura clara de conversión en una sola página.

## Contenido
- Hero con propuesta de valor y métricas
- Secciones de método, servicios y programas
- Plan semanal y testimonios
- FAQ y formulario de contacto

## Tecnologías
- HTML5
- CSS3
- JavaScript

## Estructura
- `index.html`
- `css/`
- `js/`
- `assets/`

## Uso
Abrir `index.html` en el navegador.
